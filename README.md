CyberLens Agent

Industrial QR Intelligence & Security Analysis Platform

CyberLens Agent is a cyber security platform designed to transform QR codes into actionable security intelligence. The system analyzes QR code content, extracts URLs, identifies associated brands and applications, performs threat analysis, and generates professional security reports.

Developed By: Zakia Rani

Installation

Clone Repository

git clone https://github.com/zakiarani75/CyberLens-Agent-Industrial.git

cd CyberLens-Agent-Industrial

Backend Setup

pip install -r requirements.txt

python app.py

Frontend Setup

cd frontend

npm install

npm start

---

Usage

1. Open CyberLens Agent.
2. Upload a QR Code image.
3. Click Scan QR.
4. Wait for analysis.
5. Review the generated security report.
6. Download investigation reports.
